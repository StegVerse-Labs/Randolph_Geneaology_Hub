# PATENT 04 — Self-Building Family Tree Engine

### Abstract
A computing engine that:

- Ingests external records
- Suggests lineage relationships
- Creates new family pages automatically
- Allows public publishing only with verified consent

### Claims

1. Automated profile creation via template scripting.
2. Relationship inference using census+location+DNA vectors.
3. Privacy-first — **no raw PII stored unencrypted**.
4. Supports global multi-family merging (opt-in).
