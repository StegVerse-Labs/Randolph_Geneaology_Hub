# How to Contribute

1. Fork the repo
2. Create or update files in `/profiles`, `/stories`, or `/branches`
3. Add sources or citations when available
4. Submit a Pull Request describing your update

Acceptable contributions:
- Birth/marriage/death records
- Photos, military documents, scanned letters
- DNA trace evidence & genealogical charts
- Family stories (oral histories welcome)
- Cemetery markers and obituaries
- Research leads and archival findings

Mark unverified details clearly until confirmed.
